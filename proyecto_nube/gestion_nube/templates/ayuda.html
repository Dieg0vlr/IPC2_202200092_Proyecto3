Proyecto: Sistema de Gestion en la Nube

Descripcion

Este sistema permite administrar recursos, clientes e instancias en la nube. Carga archivos XML con datos de configuracion y consumo, genera facturas y reportes en formato PDF, y analiza ventas por categoria, configuracion o recurso.

Guia rapida

  • 1. Usa "Enviar configuracion" para cargar configuracion.xml
  • 2. Usa "Enviar consumo" para cargar consumos.xml
  • 3. Presiona "Facturar" para generar facturas y PDF
  • 4. Consulta los reportes generados por categoria o recurso
